Detektiv-Agenten-Orchestrator Test Document

Dies ist ein Testdokument für die LangChain RAG Integration.

Kapitel 1: Einführung
Die Detektiv-Agenten-Orchestrator ist eine innovative Anwendung, die Cloud-basierte KI-Detektive mit lokalen KI-Zeugen kombiniert. Das System ermöglicht es, komplexe Hypothesen zu untersuchen und Widersprüche aufzudecken.

Kapitel 2: Funktionsweise
Das System arbeitet in zwei Modi:
1. AnythingLLM Modus: Verwendet vorab eingebettete Dokumente aus einem Workspace
2. LangChain RAG Modus: Verarbeitet Dokumente lokal mit Ollama und ChromaDB

Im LangChain Modus werden Dokumente automatisch in Chunks aufgeteilt, mit Embeddings versehen und in einer Vektordatenbank gespeichert. Bei Fragen werden relevante Chunks abgerufen und zur Beantwortung verwendet.

Kapitel 3: Technische Architektur
- Frontend: Electron + React + TypeScript
- Backend: Node.js mit IPC-Communication
- AI Services: Anthropic Claude, OpenAI GPT, Ollama Models
- Vector Database: ChromaDB für lokale Embeddings
- Text Processing: LangChain.js für Chunking und Retrieval

Kapitel 4: Anwendungsfälle
Dieses System eignet sich besonders für:
- Forensische Analyse von Dokumenten
- Widerspruchsaufdeckung in Texten
- Automatisierte Recherche und Validierung
- Akademische Forschung und Analyse

Kapitel 5: Zukunftsaussichten
Die Integration weiterer AI-Provider und verbesserte Retrieval-Mechanismen sind geplant. Das System soll erweitert werden um Multi-Dokument-Analysen und Kreuzvalidierung zwischen verschiedenen Quellen.
